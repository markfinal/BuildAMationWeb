Prerequisites

These are the systems that have been used during testing, with * indicating the current development configuration of the author.

Windows

  • .NET Core 2.1 SDK (includes C# Roslyn compiler)
  • Windows 7
  • Windows 8.1 *
  • Windows 10
  • VisualStudio 2017 (Community Edition) from BAM v1.1.0
  • VisualStudio 2015 (Community Edition) *
  • VisualStudio 2013 (Community Edition, preferred over Express from BAM v1.2.0+)
  • VisualStudio 2012 (Express Edition)
  • VisualStudio 2010 (Express Edition) 32-bit and 64-bit compiler support from BAM 1.2.0, no redistributable, no solution folders

macOS

  • .NET Core 2.1 SDK (includes C# Roslyn compiler)
  • macOS 10.12 (Sierra)
  • macOS 10.13 (High Sierra) *
  • Visual Studio for Mac 7.6.5+
  • Xcode 10 from BAM v2.0.0
  • Xcode 9 from BAM v1.1.4 *
  • Xcode 8 from BAM v1.1.0
  • Xcode 7
  • Xcode 6
  • Xcode 5
  • Xcode 4
