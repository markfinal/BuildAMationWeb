    Windows Development environment

    Install VisualStudio 2017 Community, from Microsoft.

    Install VisualStudio 2015 Community, Update 3, from Microsoft.

    Install VisualStudio 2013 Community, Update 5, from Microsoft.

    Install VisualStudio 2012 Express, Update 5, from Microsoft.

    Install VisualStudio 2010 Express Service Pack 1, from Microsoft.

    Linux or macOS Mono

    Install Mono for macOS or Linux (Xamarin packages recommended).

    Mono Version

    Mono 4 is the baseline version for BAM. Mono 5 is the preferred version.

    Mono IDE

    On macOS, the mono IDE is called Visual Studio for Mac.

    On Linux, the mono IDE is called MonoDevelop site.

    Historically, MonoDevelop came first, then Xamarin Studio, then Visual Studio for Mac.

    As of Mono v5, MonoDevelop on Linux can either be downloaded as a flatpak self contained archive, or will need to be compiled if flatpak is not supported on your distribution.

    GCC on Linux

    To install GCC 4.8 on CentOS 6, search the web for testing-devtools-2-centos-6.
