• Repositories

Although Bam has a default package repository with functionality to get you started, there are other software packages and tools that exist that would benefit from being included as a Bam package. Some additional package repositories exist, that offer the Bam build scripts for you to include in your own software builds.

  • bam-boost - Boost 1.60.0 & 1.64.0 scripts for building Boost dynamic libraries.
  • bam-compress - zlib 1.2.8 & 1.2.11 dynamic library build script.
  • bam-graphicssdk - Direct3D, OpenGL, GLEW, some initial support for the Vulkan SDK
  • bam-imageformats - libtiff 3.9.7, libpng 1.2.56, 1.2.57 & 1.6.26, jpeg 9b.
  • bam-parser - build scripts allowing flex and bison to be invoked, and a test case.
  • bam-python - Python 3.5.1 & Python 3.6.1 library, extension modules, shell, and test cases.
  • bam-qt - Qt 4.8.5, 5.3.2, 5.5.1, 5.6.0, 5.6.1 and 5.6.2 including the moc and rcc tools, and test cases.
  • bam-xml - TinyXML2 3.0.0 and 4.0.1 library build
  • bam-zeromq - ZeroMQ 3.2.3 and 4.1.3 library build, and test

It is encouraged that anyone can provide and publish a package repository for anyone to use.
