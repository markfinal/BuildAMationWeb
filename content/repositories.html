• Repositories

Although Bam has a default package repository with functionality to get you started, there are other software packages and tools that exist that would benefit from being included as a Bam package. Some additional package repositories exist, that offer the Bam build scripts for you to include in your own software builds.

  • bam-boost - for building Boost dynamic libraries.
  • bam-compress - zlib dynamic library build script.
  • bam-graphicssdk - Direct3D, OpenGL, GLEW, some initial support for the Vulkan SDK and MoltenVK
  • bam-imageformats - libtiff, libpng, jpeg.
  • bam-parser - build scripts allowing flex and bison to be invoked, and a test case.
  • bam-python - Python library, extension modules, shell, and test cases.
  • bam-qt - Qt 4.8.5, 5.6.2 including the moc, rcc and uic tools, and test cases.
  • bam-xml - TinyXML2 library build

It is encouraged that anyone can provide and publish a package repository for anyone to use.
